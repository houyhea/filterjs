#filterjs
##简介
一款支持过滤器功能的轻量级模版引擎。
## 原理
利用正则表达式,将模板里的变量语法替换成脚本语句,包括过滤器的语法;
本质上就是一个正则表达式的简易语法分析器.
## 语法
1. {{data.xxx|filterName}};
1. {{# statements }};
##依赖
不需要依赖其他库。
